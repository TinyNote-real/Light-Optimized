# Code of Conduct

## Our Standards

Examples of behavior that contributes to a positive environment for our
community include:

* Searching existing issues first to avoid duplicates
* Reporting bugs regarding the modpack
* Suggesting mods/changes for the modpack
* Discussing issues that are already open

Examples of unacceptable behavior include:

* Using issues for generic questions, do that in Discord instead
* Talking about issues or questions regarding hacked clients or piracy
* Disrespecting anyone
